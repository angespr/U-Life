# U-Life
Creaters: Angelina Sprague, Khanh Nguyen, Mia Chastain, and Minh Than.
## What is U-Life?
Higher Education is a stressful, complicated point of life. There is a million things to keep track of and not enough time in the day to do so. Thus, U-Life was born! Keep track of Productivity, Habits, Financial, and Opportunity all in one centralized system!

U-Life was created April 24th-26th as a part of the UWB Hackathon 2026: The Future!
### Features
- Productivity: homework tracking, todo list, pomodoro timer and google calendar integration.
- Habits: daily habit tracker, habit analytics.
- Financials: budget tracking, shared budgets, and transaction history.
- Opportunity: job board links, job application tracker, resume and cover letter hub.

### Tech Stack
- React.js
- CSS
- MongoDB

### How to use?
Up to you! Many of the features are customizable. While primarily meant for college students, U-Life can be used by all!
