# U-Life
Creaters: Angelina Sprague, Khanh Nguyen, Mia Chastain, and Minh Than.
## What is U-Life?
Higher Education is a stressful, complicated point of life. There is a million things to keep track of and not enough time in the day to do so. Thus, U-Life was born! Keep track of Productivity, Opertunity, Habits, and Financials all in one centralized system!

U-Life was created April 24th-26th as a part of the UWB Hackathon 2026: The Future!
