# Inspiration
Our inspiration came from our own experiences as students. We noticed how overwhelming college life can be when everything is spread across different platforms. We wanted to create a single space that brings all these important parts together, making it easier for students to stay organized and move closer to their goals.

# Figma
Created an illustrative demo by using Figma. Our prompt to Figma was "create a comprehensive college app with the four pages" and
Figma suggested to us the website idea that contains  
Finance page:
Monthly budgeting
Who Owes What
Recent Expenses
Upcoming Events budget

Productivity page:
Homework 
To Do (Action items)
Google Calendar integration
Pomodoro timer

Opportunities page:
More of a central dashboard of job-finding pages
LinkedIn
Handshake
Indeed
Github
Devpost
Saved Opportunities section, where students can keep track of the job application status

Daily habits page:
Customizable
Mental health check
Water intake tracker
Exercise tracker
Check box list

# Once we got the illustration from the Figma, we started the coding journey
We started the coding with these coding languages: React.js, CSS, and MongoDB.
+ React.js is used for front-end development, building component-based user interfaces for a more seamless experience.
+ CSS is used for styling and layout design.
+ MongoDB is used for storing and managing data.
